Hospital Data Analysis

A healthcare data analysis project combining Python, SQL, and Jupyter Notebook to analyze hospital data, generate insights, and visualize trends using data-driven techniques.

Project Overview

This project focuses on analyzing hospital-related datasets to better understand patient statistics, healthcare operations, and medical trends. Using Python and SQL, the project processes and visualizes healthcare data to support smarter decision-making and reporting.
